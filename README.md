# 🎵 Spotify Clone

A simple Spotify-inspired music streaming website clone built using **HTML** and **CSS**. This project recreates the user interface of Spotify's homepage, focusing on layout design, styling, and responsiveness.

## 📸 Preview

![Spotify Clone Preview](preview.png)

## 🚀 Features

- Spotify-inspired user interface
- Sidebar navigation menu
- Recently Played section
- Trending Near You section
- Featured Charts section
- Modern dark theme design
- Responsive card-based layout
- Clean and organized code structure

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🔧 How to Run the Project

1. Download or clone the repository:

```bash
git clone https://github.com/your-username/spotify-clone.git
```

2. Open the project folder.

3. Double-click on `index.html` or open it in your preferred browser.
